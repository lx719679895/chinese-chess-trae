# 中国象棋游戏 (Chinese Chess Game)

## 🎮 项目简介
这是一个功能完整、交互友好的中国象棋游戏，由 AI 助手 **Trae IDE** 为您全程生成！

本游戏基于 HTML5 Canvas 和 TypeScript 开发，提供了流畅的游戏体验和丰富的功能，适合所有喜爱中国象棋的玩家。

[🎮 在线体验](https://chesszh.netlify.app/)

## ✨ 功能特点

### 核心游戏功能
- 🎯 **完整的中国象棋规则** - 实现了所有传统中国象棋的走法和规则
- 🖱️ **直观的鼠标操作** - 点击选择棋子，点击目标位置移动

## 🛠️ 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **编程语言**: TypeScript 5.3+
- **构建工具**: Vite 5.0+

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

然后在浏览器中打开 `http://localhost:5173` 即可开始游戏。

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录中，可以部署到任何静态文件服务器上。

### 预览生产版本

```bash
npm run preview
```

### 类型检查

```bash
npm run typecheck
```

## 🎯 游戏玩法

1. **开始游戏** - 运行项目后，在浏览器中打开游戏页面
2. **选择模式** - 可以选择人机对战或双人对战
3. **移动棋子** - 点击要移动的棋子，然后点击目标位置
4. **查看提示** - 选中棋子后，会高亮显示所有合法的移动位置
5. **保存游戏** - 游戏过程中可以随时保存当前进度
6. **加载游戏** - 从保存的进度中恢复游戏

## 🤖 AI 功能

游戏内置了基于 minimax 算法的 AI 引擎，具有以下特点：

- 支持不同的搜索深度，调整 AI 难度
- 智能评估棋局，做出最优决策
- 支持悔棋功能
- 可以随时切换 AI 难度

## 🎨 自定义与扩展

### 修改棋盘和棋子样式

可以修改 `src/canvas/Board.ts` 和 `src/canvas/Piece.ts` 中的相关代码，自定义棋盘和棋子的外观。

### 扩展 AI 功能

可以修改 `src/game/AI.ts` 中的代码，优化 AI 算法或添加新的 AI 策略。

### 添加新功能

项目采用模块化设计，便于添加新功能：

- 添加音效：在相关事件中添加音频播放代码
- 添加网络对战：实现网络通信模块
- 添加游戏回放：记录和回放游戏过程
